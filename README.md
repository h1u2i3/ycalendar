# Ycalendar
A simple calendar gem for rails.
With this gem you can easily to add calendar to any model.

## Usage


## Installation
Add this line to your application's Gemfile:

```ruby
gem 'ycalendar'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install ycalendar
```
